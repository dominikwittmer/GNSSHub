# GNSSHub

GNSSHub is an open-source platform for managing GNSS RTK base stations.

It enables Raspberry Pi based GNSS receivers such as the Unicore UM980 to operate as professional RTK reference stations while providing modern monitoring, management and NTRIP services.

## Planned Features

- RTK Base Station
- NTRIP Caster
- NTRIP Client
- RTCM Monitoring
- GNSS Monitoring
- Raspberry Pi Agent
- Multi Base Station Support
- Web Dashboard
- REST API
- SignalR
- Docker
- LTE Monitoring
- MQTT
- RINEX Logging
- Remote Configuration

## Supported Hardware

- Unicore UM980
- u-blox ZED-F9P
- Raspberry Pi
- Linux
